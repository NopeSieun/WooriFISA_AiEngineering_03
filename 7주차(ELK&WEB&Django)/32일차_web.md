<br><br>

- **html 표**
  ```html
  <table>   <!-- 테이블의 시작 -->
    <thead> <!-- 제목 라인 -->
      <tr>  <!-- 행을 표현 -->
        <th></th>  <!-- 컬럼 -->
      </tr>
    </thead>
  
    <tbody> <!-- 표에 들어갈 내용 -->
      <tr>  <!-- 행을 표현 -->
        <td> </td> <!-- 컬럼 -->
      </tr>
    </tbody>
  </table>  <!-- 테이블의 끝 -->
  ```
<br>

- **html 리스트**
  ```html
  <!--리스트-->
  <ol>
      <li>1</li>
      <li>2</li>
      <li>3</li>
  </ol>
  ```
<br>

- **일반속성**: 특정 태그 안엥서만 사용 가능한 태그별 속성<br><br>
- **글로벌속성**: 모든 태그에서 사용 가능
  - class, id, 이벤트 속성, 스타일 속성 ...
